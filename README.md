## Github  Coding 同步测试


```
git init             //初始化仓库
git add -A        //更新README文件
git commit -m 'Github  Coding同步测试'     //提交更新，并注释信息
git remote set-url --add --push origin git@github.com:ProZoom/Test.git     //连接远程github项目
git remote set-url --add --push origin git@git.coding.net:ProZoom/Test.git
git push -u origin master     //将本地项目更新到github项目上去

```
